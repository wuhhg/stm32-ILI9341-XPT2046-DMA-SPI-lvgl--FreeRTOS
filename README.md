# stm32 ILI9341 + XPT2046 DMA+硬件SPI lvgl + FreeRTOS 模板

本项目基于HAL库,移植了ILI9341 + XPT2046的驱动到STM32F411CEU6上,并添加lvgl以及FreeRTOS模板

接线如图下所示.下载后默认运行lv_demo_benchmark();
